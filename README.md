Para trabajar en node.js

1-. instalar node.js de :\
https://nodejs.org/es \
2-. Crear las carpetas (Las dejé listas y ordenadas asi que no deben crearlas)\
3-. instalar las dependencias \
npm init -y \
npm install express mongoose dotenv cors \
npm install -D nodemon \
EN AMBAS CARPETAS BACKEND Y FRONTEND(ACCEDER CON CD..) INSTALAR\
**npm install** /
sino no pueden encender el servidor

4-. El archivo .env lo debe crear cada uno en la carpeta backend, con la conexion a atlas\

5.- PARA QUE PUEDAN REVISAR SI FUNCIONAN LAS CONEXIONES NECESITAN ABRIR 2 TERMINALES AL MISMO TIEMPO EN VISUAL ESTUDIO\
UNA PARA HACER ANDAR EL BACKEND Y OTRA EL FRONT, PARA  NO TENER DOS PANTALLAS DE CMD ABIERTAS PUEDEN USAR LAS TERMINALES DE VSCODE\
SI VS CODE NO LES DEJA EJECUTAR EL COMANDO **npm run dev** DEBEN ABRIR POWERSHELL, EJECUTARLO COMO ADMINISTRADOR Y POR ULTIMO EJECUTAR EL SIGUIENTE COMANDO\
**Set-ExecutionPolicy RemoteSigned -Scope CurrentUser**
SI SU PC ESTA EN ESPAÑOL DENLE **S** SI ESTA EN INGLISH **Y**\
DEBEN CERRAR EL VSCODE TOTALMENTE Y ABRIRLO DE NUEVO PARA QUE LOS CAMBIOS FUNCIONEN.
