Para trabajar en node.js

1-. instalar node.js de :\
https://nodejs.org/es \
2-. Crear las carpetas (Las dejé listas y ordenadas asi que no deben crearlas)\
3-. instalar las dependencias (creo que al yo haberlas instalado git te descarga las dependencias y te las instala de paso) si no deben ir a la carpeta del backend y ejecutar: \
npm init -y \
npm install express mongoose dotenv cors \
npm install -D nodemon \

4-. El archivo .env lo debe crear cada uno en la carpeta backend, con la conexion a atlas\
